# Problem 1:
## Description:
Given N integers, Calculate the GCD of all the numbers

## Input Format

The first line contains the length of array N
The second line contains N element representing the element of Array
Constraints

1 <= N <= 10^4
1 <= arr[i] <= 10^9
## Output Format

Output the final GCD of Number
## Sample Input 0

3
6 12 15
## Sample Output 0

3
